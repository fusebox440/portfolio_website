Project Overview

This project implements an end-to-end machine learning pipeline for sentiment analysis of tweets related to Indian elections. It includes data collection via Twitter API, preprocessing, sentiment labeling, exploratory data analysis, and deployment of deep learning models for sentiment prediction focusing on BJP and Congress.

The comprehensive workflow covers Twitter data collection using Tweepy API, data cleaning and sentiment labeling with TextBlob, exploratory data analysis with visualizations, and implementation of both GloVe word embeddings and Bidirectional LSTM models for accurate sentiment prediction.
